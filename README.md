### Pengunaan
* Clone Repository
```
git clone -b Assignment_4 https://github.com/An-Ra/Fazztrack_QA_Assigment.git
```
* Run Local Server
```
java -jar dummy-api-automation-0.0.1.jar --server.port=1234
```
* Jalankan Script
```
npm run test
```
