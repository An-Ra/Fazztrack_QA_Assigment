#### Penggunaan
* Clone Repository
```
git clone -b Assignment_3 https://github.com/An-Ra/Fazztrack_QA_Assigment.git
```
* Install Newman
```
npm install newman
```
* Menjalankan Script
```
newman run .\W2S2.postman_collection.json -e .\Environment.postman_environment.json
```
